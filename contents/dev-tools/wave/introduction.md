# Wave Terminal

Wave is a modern, open-source, and cross-platform terminal emulator.

## English Introduction

### Installation

**Windows**:
You can install Wave via `winget` or by downloading it from the official website.
```bash
winget install Wave.Wave
```

**macOS**:
You can install Wave via `brew` or by downloading it from the official website.
```bash
brew install --cask wave
```

### Key Features
*   GPU-accelerated rendering for high performance.
*   Built-in support for SSH and serial connections.
*   Tab and pane management.
