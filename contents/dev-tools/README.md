# AI Development Tools

This directory contains documentation and resources for various AI-powered development tools, organized into two main categories: IDEs and CLI tools.

## Categories

### IDEs
Collection of AI-enhanced integrated development environments that provide intelligent coding assistance:

- [Trae](./ide/trae/README.md) - AI-powered IDE with intelligent code suggestions
- [Cursor](./ide/cursor/README.md) - AI-first code editor powered by GPT-4
- [AntiGravity](./ide/antigravity/README.md) - Cutting-edge AI-powered IDE
- [Zed](./ide/zed/README.md) - High-performance, AI-powered code editor
- [VSCode](./ide/vscode/README.md) - Popular editor with AI extensions
- [CodeBuddy](./ide/codebuddy/README.md) - AI-enhanced development environment
- [Lingma (灵码)](./ide/lingma/README.md) - AI-powered IDE for Chinese-speaking developers

### CLI Tools
Command-line interfaces to various AI models for development tasks:

- [Claude Code CLI](./cli/claude-code-cli/README.md) - CLI for Anthropic's Claude AI
- [Gemini CLI](./cli/gemini-cli/README.md) - CLI for Google's Gemini AI
- [Codex CLI](./cli/codex-cli/README.md) - CLI for OpenAI's Codex model
- [Kimi CLI](./cli/kimi-cli/README.md) - CLI for Moonshot AI's Kimi model
- [GLM4 CLI](./cli/glm4-cli/README.md) - CLI for Zhipu AI's GLM4 model

## Integration Guide

For information on how to use these tools together, see the [CLI Integration Guide](./cli/README.md).

## Purpose

These tools are designed to enhance developer productivity by leveraging artificial intelligence for:
- Code generation and completion
- Error detection and debugging
- Code explanation and documentation
- Refactoring and optimization
- Natural language to code conversion
